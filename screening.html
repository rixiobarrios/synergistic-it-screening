3. What is the main difference between a computer program and computer software?

  • Computer program: A set of instructions telling a computer what to do.
  • Computer software: Programs or operations used by a computer. Computers can utilize more than one program to execute a task.

4. Describe the software development process in brief:

  • Planning:​ Decide purpose, structure, technologies to use and desired output.
  • Analysis:​ Look at the performance expectancy in various stages and make necessary modifications according to results.
  • Design:​ Use results of analysis to build the architectural structure of the project.
  • Development: ​Coding the project and its various functions.
  • Implementation:​ The application goes to a pilot program to study and see its functionalities
  • Testing:​ The software goes to a series of tests to look for errors and to debug.
  • Maintenance:​ the software is given a route to updates and scheduled revisions to keep it running properly.
  • Deployment:​ The program goes live on the internet or to the public on a package.

5. In software development process what is the meaning of debugging?

Debugging is finding error in the written code or its functionality/logic and correcting them.

8. What is the main difference between a stubs, a mock?

  • In unit testing a Stub is a ​a dummy piece of code that lets the test run, but you don't care what happens to it.
  • A mock is a dummy piece of code, that you verify is called correctly as part of the test.
